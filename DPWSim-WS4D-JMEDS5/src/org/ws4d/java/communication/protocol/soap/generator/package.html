Provides an API to generate DPWS messages out of SOAP Messages alongside message receivers and handlers, and a converse generator (from DPWS messages to SOAP)

Provides two main classes: Message2SOAPGenerator, generating SOAP messages out of DPWS messages, and its converse SOAP2MessageGenerator, generating DPWS messages out of SOAP messages.

The MessageReceiver interface enables implementing classes to receive DPWS messages created by the SOAP2MessageGenerator. The MessageHandler interface handles the received messages: each message type has its own specific handler (see EventingMessageHandler, MetadataMessageHandler, DiscoveryMessageHandler and FaultMessageHandler at org.ws4d.java.communication.protocol.soap.generator.handlers). These are both part of the SOAP2Generator API.

The UnexpectedElementException and the UnexpectedMessageException arise when an incorrect message format has been received, that is, the message is of the wrong type in the case of the first exception, or an element is present that should not be, in the case of the second exception.
